## 功能列表&&开发计划

- [x] 评论管理（增删改查）
- [x] 文章管理（增删改查）
- [x] 文章迭代栈系统（一种备份冗余机制）
- [x] 客制化主题（可用，但尚不成熟）
- [ ] 插件系统（BETA9将提供支持）
- [x] 文章聚合信息（统计功能）
- [x] 点赞
- [x] 多用户
- [ ] 用户管理（BETA9将提供支持）
- [x] 权限体系
- [x] 本地后台（jiligulu-beta）
- [x] WebApi（可选调用认证及加密访问）
- [x] 浏览统计
- [x] 并发支持（但性能上线取决于服务器硬件效能）
- [x] 插件开发（标准尚未统一）
- [x] 跨平台运行（Windows10、macOS、Linux）
- [ ] 客制化路由
